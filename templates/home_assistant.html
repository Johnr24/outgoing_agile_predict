Home Assistant

The GitHub repo includes the necessary yaml files to import the forecast into a sensor and plot on an ApexCharts card in Home Assistant.

                        sensor:
                        - platform: rest
                            resource: https://prices.fly.dev/api/G
                            scan_interval: 3600
                            name: Agile Outgoing Predict
                            value_template: {% verbatim %} "{{ value_json[0]['name']}}"{% endverbatim %} 
                            json_attributes_path: "$[0]"
                            json_attributes:
                            - "created_at"
                            - "prices"
